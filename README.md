# SystemInfo
System Audit Script (Linux)

> Скрипт выполняет полный аудит Linux-системы: собирает данные о железе, системе, сети, пользователях, открытых портах, сервисах и ключевых логах. Подходит для быстрого анализа состояния сервера или рабочей станции.

### Скрипт собирает:

1. Информация о системе

- Версия ОС
- Ядро
- Аптайм
- Переменные окружения
- Установленные обновления

2. Аппаратная часть

- CPU и количество ядер
- RAM
- Диски и разделы
- Температура (если доступно)

3. Сеть

- Интерфейсы, IP, MAC
- Маршруты
- ARP-таблица
- Активные соединения
- DNS настройки

4. Открытые порты и сервисы

- ss / netstat
- Сервисы systemd
- Активные таймеры

5. Пользователи и безопасность

- Локальные пользователи
- Sudoers
- Last logins
- Права на важные каталоги
- SSH настройки

6. Логи

- Последние ошибки системного журнала
- Аутентификация
- Ошибки ядра
- Логи SSH

### Запуск:
`chmod +x audit.sh`
`./audit.sh`

### Вывод сохраняется в файл вида:

`audit_YYYY-MM-DD_HH-MM-SS.log`

### Требования:

- Bash
- Linux (Ubuntu/Debian/CentOS/Alma/RHEL и др.)
- Права sudo для части команд

### Структура скрипта:

Скрипт разделён на функции. Основные блоки:

system_info
hardware_info
network_info
port_service_info
user_security_info
logs_info

### По завершении вся информация собирается в один отчёт.
